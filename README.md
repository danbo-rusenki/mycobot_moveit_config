# mycobot_moveit_config

## Installation

### Build from source



```sh
# download
mkdir ~p ~/mycobot_ws/src
cd ~/mycobot_ws/src
git clone https://github.com/danbo-rusenki/mycobot_moveit_config.git

# build
cd ~/mycobot_ws
colcon build --symlink-install

```

## Quick Start
```sh
source ~/mycobot_ws/install/setup.bash

```
